# Divider 分割线

区隔内容的分割线

## 基础用法

对不同段落的文本进行分割

::: demo

divider/basic

:::

## 纵向分割

通过 `direction` 属性设置分割方向

::: demo

divider/direction

:::

## 自定义间距

支持通过 `margin` 属性自由设置分割线的外边距, 单位 px

::: demo

divider/customized-margin

:::

## Divider API

### Attributes

| 属性名    | 说明           | 类型                           | 默认值     |
| --------- | -------------- | ------------------------------ | ---------- |
| direction | 分割方向       | `'horizontal'` \| `'vertical'` | horizontal |
| margin    | divider 外边距 | `number`                       | 12         |
